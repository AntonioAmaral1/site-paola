# Site da Paola

Um site especial criado em HTML, CSS e JavaScript como presente para minha namorada, contendo páginas sobre nossa história e nosso futuro juntos.

## Tecnologias Utilizadas

- **HTML5** - Estrutura do site
- **CSS3** - Estilização e responsividade
- **JavaScript** - Interatividade e animações

## 📁 Estrutura do Projeto

```
/site-paola
│── /css            # Estilos CSS
│── /images         # Imagens
│── /music          # Músicas utilizadas
│── index.html      # Página principal (Home)
│── history.html    # Página "Nossa História"
│── future.html     # Página "Nosso Futuro"
│── README.md       # Documentação do projeto
```

1. Clone o repositório:
   ```bash
   git clone https://github.com/AntonioAmaral1/site-paola.git
   ```
2. Abra o arquivo `index.html` em um navegador.
